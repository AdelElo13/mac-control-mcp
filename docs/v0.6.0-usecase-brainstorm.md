# v0.6.0 — Use-case brainstorm (multi-AI synthese)

**Doel**: een complete capability-matrix van elk "weird thing a real user
might ask their Mac agent to do". Mac-control-mcp moet elk van deze use
cases kunnen in ≤5 MCP-calls zonder handoff naar andere tools.

## Proces

Dit document groeit in rondes, elke ronde door een andere AI:

- **Ronde 1** — Claude (Sonnet 4.6): 180+ use cases uit memory + eigen
  domein-kennis van mac automation
- **Ronde 2** — Codex (gpt-5.3-codex): code-aware review; vertaalt elke
  use case naar benodigde tools, flagt gaps in implementation
- **Ronde 3** — Gemini 3.1 Pro: online research 2026 state-of-the-art,
  nieuwe macOS 16 APIs, competitor-tools, recent papers
- **Ronde 4** — Exa MCP: arxiv + github trending — wat doen academische
  papers over GUI agents recent
- **Ronde 5** — Synthese: de 4 inputs combineren, prioriteit bepalen
  (frequentie × gap × implementation effort), gap → tool mapping

Scope: "de MCP moet ALLES kunnen, Anthropic-partner niveau."
